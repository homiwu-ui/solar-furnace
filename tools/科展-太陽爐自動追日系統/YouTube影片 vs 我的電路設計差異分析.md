# YouTube 追日系統 vs 我的電路設計 — 差異分析

---

## 一、YouTube 影片簡介

**影片標題：** 超简单DIY制作太阳能光伏追光追日系统  
**來源：** https://www.youtube.com/watch?v=3Fmb7hEyB4o

這是一套極簡化的太陽能光伏追光系統，核心精神是「用最少零件、最低成本實現追日」，主要特色：
- 只用 **2 顆光敏電阻 + 比較器 + 馬達**，幾乎不需要焊接
- 不用 Arduino 或任何微控制器
- 成本壓到最低（約人民幣 10~20 元，折合台幣 50~100 元）
- 只做 **單軸追蹤**（東西向水平追蹤）

---

## 二、電路設計差異對照

| 項目 | YouTube 影片 | 我的設計（科展版） |
|:-----|:------------|:------------------|
| **追蹤軸數** | 單軸（僅東西向水平） | 雙軸（東西向 + 南北仰角） |
| **感測器** | 2 顆 LDR | 4 顆 LDR |
| **控制元件** | 比較器 IC（或電晶體直接比較） | LM393 比較器模組 |
| **馬達驅動** | 直接用比較器輸出驅動小馬達（無驅動晶片） | L293D 馬達驅動晶片（保護電路） |
| **馬達類型** | 小型 DC 馬達（無減速） | 減速馬達（1:48 或更高） |
| **電源** | 5V USB 或太陽能板 | 6V 電池組（4×AA） |
| **可變電阻** | 有（調整靈敏度） | 有（調整比較基準） |
| **LED 指示** | 無 | 有（紅/綠 LED 指示轉向） |
| **電路保護** | 無（電流直接流過 IC） | L293D 有過熱保護 |
| **成本** | 約 NT$ 50~100 | 約 NT$ 350~500 |
| **難度** | 極簡單（20 分鐘可完成） | 中等（需要了解電路原理） |

## 三、YouTube 影片電路原理（簡化版）

```
          5V
           │
       ┌───┴───┐
       │ 開關   │
       └───┬───┘
           │
     ┌─────┼─────────────────────────────────┐
     │     │                                  │
   ┌─┴─┐ ┌┴┐                              ┌──┴──┐
   │LDR│ │R│                              │LDR  │
   │左 │ │ │                              │右   │
   └─┬─┘ └┬┘                              └──┬──┘
     │     │                                  │
     │     │                                  │
     │  ┌──┴───────────────────────────────┐  │
     │  │        比較器 (或電晶體)          │  │
     │  │   當左光強→右LDR電壓低→馬達正轉  │  │
     │  │   當右光強→左LDR電壓低→馬達反轉  │  │
     │  └────────────────┬────────────────┘  │
     │                   │                    │
     │              ┌────▼────┐               │
     │              │ 小DC馬達 │               │
     │              │ (無減速) │               │
     │              └─────────┘               │
     │                                        │
     └────────────────────────────────────────┘
```

**最大特點：**
- 沒有 L293D 馬達驅動，比較器直接輸出給馬達
- 沒有減速齒輪，馬達轉速快但扭力小
- 只做水平追蹤，不追仰角
- 非常便宜，但精度有限

## 四、YouTube 影片的機械結構

| 項目 | YouTube 影片 | 我的設計（科展版） |
|:-----|:------------|:------------------|
| **轉動載臺** | 簡易圓盤（可能用紙板或塑膠） | 木製底座 + 水平轉盤 + 滾珠 |
| **馬達安裝** | 直接黏在底座 | 齒輪組減速（10齒→60齒） |
| **軸承** | 無（直接摩擦轉動） | 4 顆滾珠軸承減少摩擦 |
| **太陽爐** | 平面光伏板（不需要聚焦） | 拋物面鏡太陽爐（需要精確聚焦） |
| **負載重量** | 約 100~200g（光伏板） | 約 500~1000g（壓克力鏡面爐） |

## 五、為什麼差異這麼大？— 設計目標不同

| 目標 | YouTube 影片 | 我的設計 |
|:-----|:------------|:---------|
| **用途** | 光伏板追光（只要對著光就行） | 太陽爐聚焦（焦點偏移 1mm 就沒效率） |
| **精度要求** | 低（±5度可接受） | 高（±1度以內） |
| **負載重量** | 輕（光伏板） | 重（18 片壓克力鏡面） |
| **追蹤軸數** | 單軸夠用 | 必須雙軸（仰角影響聚焦） |
| **成本限制** | 越便宜越好 | 科展可接受合理成本 |

## 六、YouTube 影片電路的優缺點分析

### 優點 ✅
1. **成本極低**：約 NT$ 50~100 即可完成
2. **製作快速**：30 分鐘內可組裝完成
3. **原理簡單**：國小生也能理解「光強不同→電壓不同→馬達轉動」
4. **適合光電板追蹤**：光伏板對角度精度要求不高，單軸追蹤即可提升 20~30% 效率

### 缺點 ❌
1. **無減速機構**：小 DC 馬達轉速太快（數百 RPM），會造成過衝（overshoot）震盪
2. **無驅動保護**：比較器直接接馬達，大電流可能燒毀 IC
3. **無垂直追蹤**：太陽仰角變化大（冬夏可差 30 度），單軸不夠
4. **精度不足**：太陽爐需要精確聚焦，這種設計無法維持焦點
5. **無負載能力**：帶不動 1kg 以上的太陽爐

## 七、我的設計改良重點

針對 YouTube 影片的不足，我的科展版做了以下改良：

| 問題 | 改良方案 | 效果 |
|:-----|:---------|:-----|
| 轉速太快 | 加入減速馬達（1:48 以上） | 轉速降至 200 RPM，更穩定 |
| 扭力不足 | 齒輪二次減速（6:1） | 扭力放大 6 倍，帶動太陽爐 |
| 無驅動保護 | 加入 L293D 馬達驅動 IC | 過熱保護、H 橋正反轉控制 |
| 只有水平追蹤 | 加入垂直仰角追蹤 | 雙軸同步，全天候追蹤 |
| 摩擦力大 | 底座加滾珠軸承 | 摩擦力降至 1/10 |
| 無狀態指示 | 加入紅綠 LED | 一目了然知道馬達轉向 |

## 八、建議：科展可以用哪個版本？

| 版本 | 適合對象 | 預算 | 建議 |
|:-----|:---------|:----:|:-----|
| **YouTube 簡化版** | 低年級、初步體驗 | NT$ 50~100 | 可先做這個驗證概念，再升級 |
| **科展完整版** | 中高年級、正式比賽 | NT$ 350~500 | 正式科展建議用這套 |
| **Arduino 智慧版** | 國中以上、想挑戰程式 | NT$ 500~800 | 可加入 PID 控制、時間預測 |

### 升級路徑建議
```
YouTube 簡化版（驗證概念）
    │
    ├── ① 加入減速馬達（解決轉速問題）
    │
    ├── ② 加入 L293D（解決驅動保護問題）
    │
    ├── ③ 加入垂直追蹤（升級為雙軸）
    │
    ├── ④ 加入滾珠載臺（解決摩擦問題）
    │
    └── ⑤ 升級為科展完整版
```

## 九、總結

**YouTube 影片是一套「夠用就好」的極簡設計**，適合快速體驗太陽能追光概念，但不適合精確的太陽爐聚焦應用。

**我的科展版是「完整嚴謹」的設計**，針對太陽爐聚焦的高精度需求，加入了減速、雙軸、驅動保護、滾珠載臺等改良，成本較高但效果顯著。

兩者的核心原理相同（光敏電阻比較→馬達驅動），差異在於**精度、負載能力、保護機制**。建議科展從簡化版開始理解原理，再逐步升級到完整版。
