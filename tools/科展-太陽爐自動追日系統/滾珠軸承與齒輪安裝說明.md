# 滾珠軸承安裝 & 齒輪傳動 — 詳細說明

---

## 一、為什麼需要滾珠軸承？

太陽爐重約 1 公斤，如果直接放在木板上轉動：
- 木頭摩擦力大 → 馬達帶不動
- 轉不順 → 震動 → 焦點跑掉

**解決方法：** 用滾珠把「滑動摩擦」變成「滾動摩擦」，摩擦力立刻降為 1/10！

```
  沒有滾珠（滑動摩擦）：        有滾珠（滾動摩擦）：

  ┌─────────────┐              ┌─────────────┐
  │  太陽爐轉盤  │              │  太陽爐轉盤  │
  └──────┬──────┘              └──┬───┬───┬──┘
         │                        ●   ●   ●  ← 滾珠
    ═════╧═════                ═══════════════
       底座                        底座

  摩擦力：大（卡卡的）          摩擦力：小（滑順的）
  馬達：很吃力                  馬達：輕鬆轉
```

---

## 二、滾珠軸承規格選擇

| 項目 | 建議規格 | 說明 |
|:-----|:---------|:-----|
| **軸承型號** | 608ZZ 或 625ZZ | 最常見的規格，便宜又好買 |
| **外徑** | 22mm（608）或 16mm（625） | 太大不好嵌入木板 |
| **內徑** | 8mm（608）或 5mm（625） | 要配合你的轉軸直徑 |
| **厚度** | 7mm（608）或 5mm（625） | — |
| **數量** | 4 顆 | 放在轉盤四角 |
| **參考價格** | NT$ 15~25 / 顆 | 蝦皮或五金行 |
| **購買關鍵字** | 「608ZZ 軸承」或「溜冰鞋軸承」 | 溜冰鞋用的就是這種！ |

> 💡 **小秘密：** 你家如果有舊的直排輪或溜冰鞋，拆下來的軸承就可以用！

## 三、滾珠軸承安裝步驟（一步一步教）

### 準備工具
- 電鑽（鑽孔用）
- 鑽頭（直徑比軸承外徑大 0.5mm）
- 熱熔膠槍
- 尺、鉛筆
- 砂紙

### Step 1：在轉盤上標記位置

把轉盤（25cm × 25cm 木板）翻到背面，在四個角落各標記一個點。

```
  轉盤背面（上視圖）：

    ┌─────────────────────────┐
    │                         │
    │  ⊕ ← A點          ⊕ ← B點   │
    │  (距邊2cm)        (距邊2cm)  │
    │                         │
    │         中心點⊕          │
    │                         │
    │  ⊕ ← C點          ⊕ ← D點   │
    │  (距邊2cm)        (距邊2cm)  │
    │                         │
    └─────────────────────────┘

  標記位置：距邊緣 2~3 cm 的地方
```

### Step 2：鑽孔

```
  鑽孔示意圖：

  鑽頭直徑 = 軸承外徑 + 0.5mm

  例如：608ZZ 外徑 22mm → 用 22.5mm 鑽頭
       625ZZ 外徑 16mm → 用 16.5mm 鑽頭

  鑽孔深度 = 軸承厚度（約 5~7mm）
  ⚠️ 不要鑽穿！留至少 3mm 在底部

    木板側面圖：

    ┌──────────┐
    │          │ ← 鑽孔深度 5~7mm
    │  ┌────┐  │
    │  │軸承│  │ ← 嵌入軸承
    │  └────┘  │
    │    ██████│ ← 底部留 3mm 以上
    └──────────┘
```

### Step 3：嵌入軸承

把軸承放進鑽好的孔裡：
- 軸承應該要「緊緊的」卡在孔裡（稍緊是正常的）
- 如果太緊，用砂紙稍微磨一下孔壁
- 如果太鬆，下面會教怎麼補救

```
  嵌入軸承：

    放入前：            放入後：
    ┌──────────┐       ┌──────────┐
    │  ○○○○○○  │       │  ○○○○○○  │ ← 軸承頂面和木板齊平
    │          │       │ ╔══════╗ │
    │   空孔   │  →    │ ║ 軸承 ║ │
    │          │       │ ╚══════╝ │
    │    ████  │       │    ████  │
    └──────────┘       └──────────┘
```

### Step 4：固定軸承

用熱熔膠在軸承四周固定（不要滴到軸承裡面！）：

```
  熱熔膠位置（截面圖）：

    ┌──┬────┬──┐
    │  │軸承│  │
    │  │    │  │
    │  └────┘  │
    │ ▓▓    ▓▓│ ← 熔膠在軸承外圈和木頭之間
    │    ████  │
    └──────────┘

    ⚠️ 不要把膠滴到軸承的滾珠上！
    ⚠️ 軸承的內圈（中心圓孔）要保持乾淨
```

### Step 5：安裝轉軸

在底座中心鑽一個孔，裝上一根金屬桿（轉軸），穿過轉盤中心。

```
  轉軸安裝（側面圖）：

    ┌─────────────────────┐
    │      太陽爐轉盤      │
    │   ⊕      ⊕      ⊕   │ ← 軸承
    └──────────┬──────────┘
               │ ← 轉軸（金屬桿，直徑 8mm 或 5mm）
    ┌──────────┴──────────┐
    │        底座          │
    └─────────────────────┘

    轉軸上端：穿過轉盤中心的孔，用 C 型扣或螺帽固定
    轉軸下端：固定在底座上，不能動
```

### Step 6：測試轉動

把轉盤放到軸承上，用手指輕輕推一下：
- ✅ 轉盤應該會滑順轉動 3~5 圈才停
- ❌ 如果卡卡的 → 檢查軸承有沒有裝正、孔有沒有太小
- ❌ 如果太晃 → 檢查軸承有沒有太鬆（下面教補救方法）

### Step 7：太鬆的補救方法

如果鑽孔太大、軸承會晃：
1. 在軸承外圈和孔壁之間塞入薄木片或紙板
2. 再用熱熔膠固定
3. 或者用「環氧樹脂」（AB 膠）填補縫隙

## 四、齒輪傳動系統說明

### 4.1 為什麼要用齒輪？

馬達轉太快了！TT 馬達 1:48 減速後還是 200 RPM（每分鐘轉 200 圈）。太陽每分鐘才移動 0.25 度，我們需要更慢！

**解決方法：** 再加一組齒輪做二次減速，把 200 RPM 降到約 33 RPM。

```
  齒輪減速原理：

  小齒輪（10 齒）──→ 大齒輪（60 齒）

  ┌──┐
  │●●│ ← 小齒輪（10 齒，直徑 7cm）
  └──┘
     ┌────────────────────────┐
     │●  ●  ●  ●  ●  ●  ●  ●  ●│ ← 大齒輪（60 齒，直徑 35cm）
     │●                      ●│
     └────────────────────────┘

  減速比 = 大齒輪 ÷ 小齒輪 = 60 ÷ 10 = 6:1

  意思是：小齒輪轉 6 圈，大齒輪才轉 1 圈
  馬達 200 RPM → 經過 6:1 減速 → 轉盤 ≈ 33 RPM
  再經過 TT 馬達內部 1:48 → 最終 ≈ 4 RPM
```

### 4.2 齒輪規格

| 項目 | 小齒輪（主動） | 大齒輪（從動） |
|:-----|:--------------:|:--------------:|
| 齒數 | 10 齒 | 60 齒 |
| 直徑 | 約 7cm | 約 35cm |
| 減速比 | — | 6:1 |
| 材質 | 木合板 或 塑膠 | 木合板 |
| 厚度 | 1.2cm | 1.2cm |
| 製作方式 | 齒輪產生器 + 線鋸機 | 齒輪產生器 + 線鋸機 |

### 4.3 自己做木齒輪（步驟）

#### Step 1：用軟體產生齒輪模板

免費網站：http://woodgears.ca/gear/index.html

```
  齒輪模板產生器操作：
  1. 打開網站
  2. 輸入參數：
     - Number of teeth（齒數）：10（小齒輪）或 60（大齒輪）
     - Pitch diameter（直徑）：7cm 或 35cm
     - Pressure angle（壓力角）：20 度（預設就好）
  3. 按「Generate」產生圖形
  4. 列印出來
```

#### Step 2：貼到木板上裁切

```
  製作流程：

  列印模板    貼到木板    鑽中心孔    線鋸切齒
     │           │           │           │
     ▼           ▼           ▼           ▼
  ┌─────┐   ┌─────┐    ┌─────┐    ┌─────┐
  │📄   │ → │木+📄│ →  │木+📄│ →  │     │
  │模板 │   │     │    │  ○  │    │ ⚙️  │
  └─────┘   └─────┘    └─────┘    └─────┘

  ⚠️ 線鋸機很鋒利，要大人操作！
```

#### Step 3：齒輪嚙合測試

把小齒輪和大齒輪放在一起，用手轉動：
- ✅ 齒與齒之間要能順暢卡合
- ❌ 如果卡住 → 用砂紙磨一下齒的邊緣
- ❌ 如果太鬆 → 齒的間距可能不對，重新製作

```
  齒輪嚙合示意圖：

       小齒輪
       ┌──┐
       │●●│ ← 轉動
       └┬┬┘
        ││
     ┌──┘└──────────────────────┐
     │●  ●  ●  ●  ●  ●  ●  ●  ●│ ← 大齒輪
     │●  ●  ●  ●  ●  ●  ●  ●  ●│    （跟著轉，但比較慢）
     └──────────────────────────┘

  小齒輪轉 6 圈 = 大齒輪轉 1 圈
```

## 五、馬達和齒輪的組合安裝

### 5.1 齒輪安裝位置

```
  水平傳動安裝圖（俯視圖）：

    ┌─────────────────────────────────┐
    │                                 │
    │         大齒輪（60齒）           │
    │        ╱            ╲          │
    │       ╱   轉盤中央    ╲         │
    │      ╱                  ╲       │
    │     ╱                    ╲      │
    │    ╱______________________╲     │
    │           ↑                   │
    │     ┌─────┴─────┐             │
    │     │  小齒輪   │             │
    │     │  (10齒)   │ ← 馬達軸心   │
    │     └─────┬─────┘             │
    │           │                     │
    │     ┌─────┴─────┐             │
    │     │  減速馬達  │             │
    │     └───────────┘             │
    │           ↑                     │
    │      固定在底座上                │
    └─────────────────────────────────┘
```

### 5.2 安裝步驟

#### Step 1：固定大齒輪
把大齒輪固定在轉盤的底部中央：
- 大齒輪中心鑽一個孔，穿過轉盤
- 用螺絲鎖緊，讓大齒輪和轉盤一體轉動

#### Step 2：固定小齒輪和馬達
把小齒輪裝在馬達軸心上，再把馬達固定在底座：
- 小齒輪用熱熔膠或螺絲固定在馬達軸心上
- 馬達用 L 型鐵架或木塊固定在底座上
- 調整位置，讓小齒輪和大齒輪正好卡合

#### Step 3：調整間距

```
  齒輪間距調整（側面圖）：

  正確間距：            太近：              太遠：
  ┌──┐                  ┌──┐                ┌──┐
  │●●│                  │●●│                │●●│
  └┬┬┘                  └┬┬┘                └┬┬┘
   ││ ← 1~2mm 間隙      │×│ 卡住了！        │  │ 沒嚙合！
  ┌┘└──┐              ┌─┘└─┐             ┌──┘└──┐
  │●●●●│              │●●●●│             │●●●●●●│
  └────┘              └────┘             └──────┘

  ✅ 齒與齒之間留 1~2mm 的間隙
  ⚠️ 間隙太小會卡住，太大會打滑
```

#### Step 4：測試
- 手動轉動小齒輪，大齒輪要跟著轉
- 接上電池，馬達轉動時要能帶動大齒輪
- 如果轉不動 → 檢查間距、齒輪有沒有卡到東西

## 六、垂直仰角的齒輪安裝

垂直方向（仰角）也需要一組齒輪，原理相同但結構不同。

```
  垂直傳動安裝圖（側面圖）：

    ┌─────────────────┐
    │     太陽爐       │
    │    ╱    ╲        │
    │   ╱  ⭐  ╲       │ ← 焦點
    │  ╱________╲      │
    │       │          │
    │  ┌────┴────┐     │
    │  │ 垂直轉軸 │     │ ← 太陽爐繞這根軸上下轉
    │  └────┬────┘     │
    │       │          │
    │  ┌────┴────┐     │
    │  │ 大齒條  │ ← 齒條（直線型齒輪）
    │  └────┬────┘     │
    │       │          │
    │  ┌────┴────┐     │
    │  │ 小齒輪  │ ← 馬達軸心
    │  └────┬────┘     │
    │  ┌────┴────┐     │
    │  │ 垂直馬達 │     │
    │  └─────────┘     │
    │       │          │
    │  ┌────┴────┐     │
    │  │ 水平轉盤 │     │
    │  └─────────┘     │
    └─────────────────┘

  垂直方向用「齒條」而不是圓形齒輪
  齒條就是一條直線的齒輪，像鋸子一樣
```

### 齒條製作
1. 用 GeoGebra 畫出齒條模板
2. 貼到 1.2cm 木板上
3. 用線鋸機裁切齒形
4. 固定在太陽爐側邊的支架上

### 齒條安裝位置

```
  齒條安裝（正面圖）：

       太陽爐側面
    ┌─────────────┐
    │             │
    │    太陽爐    │
    │             │
    │  ┌──┐       │
    │  │齒│ ← 齒條固定在側邊
    │  │條│       │
    │  │  │       │
    │  └┬─┘       │
    │   │         │
    │  ┌┴──┐      │
    │  │小齒│ ← 垂直馬達帶動小齒輪
    │  │輪  │     │
    │  └───┘      │
    └─────────────┘

  馬達轉 → 小齒輪轉 → 齒條上下移動 → 太陽爐仰角改變
```

## 八、調整與除錯

| 問題 | 原因 | 解決方法 |
|:-----|:-----|:---------|
| 轉盤轉不動 | 軸承沒裝正 | 拆下來重新嵌入，確保頂面與木板齊平 |
| 轉盤很晃 | 軸承孔太大 | 塞木片或紙板填補縫隙，再用膠固定 |
| 齒輪卡住 | 間距太小 | 往外移一點，留 1~2mm 間隙 |
| 齒輪打滑 | 間距太大 | 往內移一點，或加一片薄木片墊高 |
| 馬達空轉 | 小齒輪沒固定緊 | 用熱熔膠重新固定在馬達軸心上 |
| 轉動有聲音 | 齒輪摩擦 | 用砂紙磨平齒的邊緣，或加一點蠟 |
| 垂直方向太重 | 沒有平衡 | 在太陽爐對側加配重（如螺絲） |

## 九、安全提醒

1. **鑽孔時**：一定要有大人陪同，戴護目鏡
2. **熱熔膠**：剛擠出的膠很燙（約 200℃），避免碰到皮膚
3. **線鋸機**：鋸齒輪時慢慢鋸，不要急，手指離鋸條 5cm 以上
4. **測試時**：接電池前先檢查接線有沒有短路，避免燒壞零件
5. **搬運時**：太陽爐的鏡面會反射強光，搬運時不要對著人的眼睛
