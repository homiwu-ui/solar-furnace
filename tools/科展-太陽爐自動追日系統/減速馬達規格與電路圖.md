# 減速馬達規格型號 & 完整電路圖

---

## 一、為什麼需要「減速」馬達？

太陽在天空每小時移動約 15 度，每分鐘僅移動 **0.25 度**。如果直接用一般直流馬達（轉速 2000~5000 RPM）：
- 太快！太陽爐一轉就過頭
- 扭力太小！轉不動

**解決方案：** 加減速齒輪箱 → 轉速變慢、扭力放大

---

## 二、推薦型號比較

### 首選：TT 減速馬達（1:48）
最便宜、最容易買到，非常適合科展

| 項目 | 規格 |
|:-----|:------|
| **型號** | TT 馬達（也稱智慧小車馬達） |
| **電壓** | DC 3V~6V |
| **減速比** | 1:48 |
| **空載轉速** | 200 RPM（@6V）/ 90 RPM（@3V） |
| **空載電流** | ≤200mA（@6V） |
| **扭力** | 約 0.5 kg·cm（經 1:48 放大後） |
| **參考價格** | NT$ 35~45 |
| **購買地點** | 光華商場、蝦皮、露天、台灣物聯科技 |
| **適用** | 搭配外部齒輪再減速（6:1），最終轉速約 33 RPM，可採間歇驅動 |

**優點：** 便宜、好買、體積小、有塑膠齒輪箱  
**缺點：** 塑膠齒輪不耐重壓、轉速仍偏高（須用軟體或電路做間歇控制）

---

### 進階：N20 全金屬減速馬達
扭力更大、更耐用

| 項目 | 規格 |
|:-----|:------|
| **型號** | GA12-N20 微型金屬減速馬達 |
| **電壓** | DC 3V~12V |
| **減速比** | 多種可選：1:30、1:50、1:100、1:150、1:210、1:298 |
| **空載轉速** | 依型號：30 / 50 / 100 / 150 / 210 RPM（@6V） |
| **扭力** | 約 0.8~2.5 kg·cm（依減速比而定） |
| **參考價格** | NT$ 60~120 |
| **購買地點** | 台灣智能感測科技、蝦皮、露天 |

**推薦型號：** GA12-N20 6V **30RPM**（減速比約 1:150）  
→ 搭配轉盤 6:1 後，最終轉速約 5 RPM，間歇驅動效果很好

---

### 進階：強扭力直流減速馬達（2 RPM）
轉速極慢、扭力超大，最適合追日

| 項目 | 規格 |
|:-----|:------|
| **型號** | 強扭力直流減速馬達 6V 2RPM（金順/源頭廠商） |
| **電壓** | DC 6V |
| **減速比** | 約 1:1000 |
| **空載轉速** | **2 RPM** |
| **扭力** | 約 5~8 kg·cm（非常大） |
| **參考價格** | NT$ 150~250 |
| **購買地點** | 金順電子、今華電子、蝦皮 |

**優點：** 轉速極慢、扭力巨大，甚至可以直接驅動轉盤不需額外齒輪  
**缺點：** 較貴、體積較大、較重

---

### 三款比較總表

| 項目 | TT 馬達 1:48 | N20 30RPM | 強扭力 2RPM |
|:-----|:------------:|:---------:|:-----------:|
| 價格 | NT$ 35~45 | NT$ 60~120 | NT$ 150~250 |
| 轉速 (@6V) | 200 RPM | 30 RPM | 2 RPM |
| 扭力 | 中 | 中高 | 高 |
| 外部齒輪 | 需要（6:1以上） | 建議（3:1） | 可省略 |
| 適合對象 | 預算有限 | 最佳平衡 | 效果最好 |
| **科展推薦** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

## 三、馬達動力夠不夠支撐太陽爐？— 具體計算

### 扭力驗算

```
假設：
- 太陽爐重量：約 1 kg
- 支點到重心距離：約 10 cm
- 所需扭矩 = 1 kg × 10 cm × (1/6齒輪比) = 1.67 kg·cm

TT 馬達（1:48）經 6:1 減速後扭力放大 6 倍：
→ 有效扭力 ≈ 1.67 × 6 = 10 kg·cm  ✓ 夠用

N20（30RPM）經 3:1 減速後：
→ 有效扭力 ≈ 2.5 × 3 = 7.5 kg·cm  ✓ 夠用

強扭力 2RPM 直接驅動：
→ 扭力 8 kg·cm  ✓ 夠用
```

> **結論：三款都夠用。** 只要有齒輪減速放大扭力，小馬達也能輕鬆推動太陽爐。

### 4.3 LM393 比較器模組接線（水平感測，垂直相同）

```
          5V (+) ─────┬───────┐
                      │       │
                    ┌─┴─┐   ┌─┴─┐
                    │LDR│   │LDR│       ← 光敏電阻（左右兩側）
                    │左 │   │右 │
                    └─┬─┘   └─┬─┘
                      │       │
                    ┌─┴─┐   ┌─┴─┐
                    │10K│   │10K│       ← 分壓電阻（下拉至 GND）
                    └─┬─┘   └─┬─┘
                      │       │
          V+ ────────┤       ├──────── V-      ← 接 LM393 Pin3 & Pin2
                      │       │
                   ┌──┴───────┴──┐
                   │   LM393     │
                   │   比較器     │
                   │             │
                   │ Pin1 = OUT  ├──────→ 去 L293D 輸入
                   │ Pin8 = VCC  ├──────→ 5V
                   │ Pin4 = GND  ├──────→ GND
                   └─────────────┘

    LM393 腳位對照：
    ┌──────────────────────────────────┐
    │  ┌──┐                            │

    │  │3 │  V+ A          V- B │6│   │
    │  │4 │  GND           V+ B │5│   │
```

### 4.4 L293D 馬達驅動電路接線

```
               L293D 腳位
    ┌──────────────────────────────────┐
    │  ┌──┐                            │
    │  │1 │  Enable 1        VCC │16│  ├── 5V (邏輯電源)
    │  │2 │  Input 1       Input 4│15│
    │  │3 │  Output 1      Output 4│14│
    │  │4 │  GND             GND │13│  ├── GND
    │  │5 │  GND             GND │12│  ├── GND
    │  │6 │  Output 2      Output 3│11│
    │  │7 │  Input 2       Input 3│10│
    │  │8 │  VS (馬達電源)     Enable 2│9│  ├── 6V (馬達電源)
    │  └──┘                            │
    └──────────────────────────────────┘

    實際接線：
    ┌──────┐
    │LM393 ├──── Pin 1 (OUT) ────→ L293D Pin 2 (Input 1)
    │水平   │                      L293D Pin 7 (Input 2) → GND (固定)
    │OUT   │
    └──────┘

    L293D Pin 1 (Enable 1) ────→ 5V (永遠致能)
    L293D Pin 3 (Output 1) ────→ 馬達紅線 (+)
    L293D Pin 6 (Output 2) ────→ 馬達黑線 (-)
    L293D Pin 8 (VS) ────────→ 6V 電池
    L293D Pin 16 (VCC) ──────→ 5V (內建穩壓)
    L293D Pin 4,5 (GND) ─────→ GND

    運作邏輯：
    LM393 OUT = 高電位（V+ > V-）→ 馬達正轉（向東/向上）
    LM393 OUT = 低電位（V+ < V-）→ 馬達反轉（向西/向下）
    LM393 OUT = 中間（V+ = V-）→ 馬達停止（已對正）
```

## 五、運作流程圖

```
 ┌──────────────────────────┐
 │ 系統啟動（開關 ON）      │
 │ LED 亮 --> 電源正常       │
 └──────────┬───────────────┘
            ▼
 ┌──────────────────────────┐
 │ 水平 LDR 讀取            │ ← 偵測東西向光強
 │ V+ (左) vs V- (右)       │
 └──────────┬───────────────┘
            ▼
    ┌───────┴───────┐
    │ V+ = V- ?      │──── YES ──→ 馬達停止（已對正）
    └───────┬───────┘
           NO
            ▼
    ┌───────┴───────┐
    │ V+ > V- ?      │──── YES ──→ 馬達正轉（向東）
    └───────┬───────┘
           NO
            ▼
    ┌──────────────────────────┐
    │ 馬達反轉（向西）          │
    └──────────┬───────────────┘
               ▼
    ┌──────────────────────────┐
    │ 重新讀取 LDR            │ ← 回到判斷迴圈
    │ 直到 V+ = V- 才停止      │
    └──────────────────────────┘
               │
    ┌──────────▼───────────────┐
    │ 垂直 LDR 同步進行（同上） │ ← 南北向仰角調整
    │ 兩軸獨立控制               │
    └──────────────────────────┘
               │
    ┌──────────▼───────────────┐
    │ 每 1~2 秒重覆掃描一次     │ ← 持續追蹤
    │ 太陽移動就自動修正        │
    └──────────────────────────┘
```
