# 太陽爐自動追日系統 — 硬體製作方法

---

## 一、整體架構

```
        ┌─────────────────────────┐
        │      拋物面太陽爐        │  ← 18片鏡面壓克力組合
        └──────────┬──────────────┘
                   │
        ┌──────────▼──────────────┐
        │    鍋具支架（焦點處）    │  ← 距爐底 10cm
        └──────────┬──────────────┘
                   │
        ┌──────────▼──────────────┐
        │   垂直轉動軸（南北向）   │  ← 減速馬達①控制仰角
        └──────────┬──────────────┘
                   │
        ┌──────────▼──────────────┐
        │   水平轉動盤（東西向）   │  ← 減速馬達②控制方位角
        │     + 滾珠 × 4          │
        └──────────┬──────────────┘
                   │
        ┌──────────▼──────────────┐
        │       木製底座           │
        └─────────────────────────┘
```

## 二、太陽爐本體製作（拋物面鏡）

### 材料
- 鏡面壓克力板 2mm × 18 片（每片尺寸依設計圖裁切）
- 銅片（寬 1cm）× 36 片（用於連結相鄰爐片）
- 螺絲 + 螺帽 × 36 組
- 熱熔膠條 × 5 支

### 工具
- 線鋸機（或手工線鋸）
- 烤箱（可達 115℃）
- 熱熔槍
- 螺絲起子
- 砂紙（打磨邊緣）

### 製作步驟

#### Step 1：繪製單片模型
1. 開啟 GeoGebra，輸入拋物線 \( y = \frac{1}{40}x^2 \)（焦距 10cm）
2. 以 x = 6、8、10、12、14、16、18、20cm 為半徑計算弧長
3. 將拋物線分為 18 等份，繪出單片展開圖並列印

#### Step 2：製作模具
1. 將展開圖貼於木合板上
2. 用線鋸機沿輪廓裁切，做出模具
3. 用砂紙磨平邊緣

#### Step 3：壓克力板裁切
1. 將鏡面壓克力板放在模具上劃線
2. 用線鋸機裁切 18 片
3. 用砂紙打磨邊緣毛刺

#### Step 4：壓克力彎曲定型
1. 將單片壓入模具
2. 放入烤箱以 115℃ 加熱 6.5 分鐘
3. 取出後冷卻 10 分鐘定型，即得拋物線形單片

#### Step 5：組裝組合
1. 將 18 片開口端朝上，相鄰兩片以銅片 + 螺絲鎖合
2. 底端（尖端）以熱熔膠固定
3. 檢查是否有縫隙，以熱熔膠補齊

### 注意
- 鏡面壓克力有保護膜，組裝完成後再撕除
- 每片之間盡量密合，減少光線漏失

## 三、轉動載臺製作

### 材料
- 木合板（30cm × 30cm × 1.5cm）× 2 片（底座 + 轉盤）
- 木條（2cm × 4cm × 20cm）× 4 根
- 滾珠軸承（直徑 1cm）× 4 顆
- 減速馬達 6V × 2 顆
- 齒輪組（10 齒 + 60 齒）× 2 組
- L 型鐵架 × 4 個
- 螺絲組 × 1 包

### 製作步驟

#### Step 1：底座製作
1. 裁切 30cm × 30cm 木合板作為底座
2. 在四角鎖上 L 型鐵架增加穩定性

#### Step 2：水平轉動盤
1. 裁切 25cm × 25cm 木合板作為轉盤
2. 在轉盤底部四角嵌入滾珠軸承
3. 將轉盤放到底座上，確認轉動順暢

#### Step 3：水平傳動機構
1. 將 60 齒大齒輪固定在轉盤中心下方
2. 將 10 齒小齒輪安裝在減速馬達① 的輸出軸
3. 將馬達固定在底座上，使兩齒輪嚙合
4. 小齒輪帶動大齒輪，減速比 6:1

#### Step 4：垂直轉動機構
1. 在轉盤兩側豎立兩根木條（間距約 20cm）
2. 頂端鑽孔作為太陽爐的支點
3. 安裝減速馬達② 於一側木條
4. 馬達軸心連接太陽爐側邊的齒輪
5. 太陽爐背面也用木條做支撐，形成三角形的穩定結構

#### Step 5：組裝測試
1. 用手轉動確認水平、垂直轉動皆順暢
2. 接上 6V 電池測試馬達正反轉
3. 調整齒輪嚙合間隙，避免卡死

## 四、追日電路製作

### 電路原理

```
        光敏電阻1 ─┬─ V+
                    │
        光敏電阻2 ─┴─ V-
                        │
                   ┌────▼────┐
                   │ LM393   │
                   │ 比較器   │
                   └────┬────┘
                        │ 輸出訊號
                        │
              ┌─────────▼─────────┐
              │   L293D 馬達驅動   │
              │   (或繼電器模組)   │
              └─────────┬─────────┘
                        │
              ┌─────────▼─────────┐
              │   減速馬達 ±6V    │
              └───────────────────┘
```

### 材料

| 零件 | 數量 | 備註 |
|------|:----:|------|
| 光敏電阻（LDR 5528） | 4 顆 | 水平 2 顆、垂直 2 顆 |
| LM393 比較器模組 | 2 塊 | 水平、垂直各 1 塊 |
| L293D 馬達驅動晶片 | 2 顆 | 或使用繼電器模組 |
| 可變電阻 10KΩ | 2 顆 | 調整靈敏度 |
| 電阻 10KΩ | 4 顆 | 分壓用 |
| 電阻 330Ω | 4 顆 | LED 限流 |
| LED（紅、綠） | 各 2 顆 | 指示電源與轉向 |
| 麵包板 | 1 塊 | 電路測試用 |
| 杜邦線 | 1 包 | 接線用 |
| 6V 電池盒（4 顆 AA） | 1 個 | 電源 |
| 開關 | 1 個 | 電源開關 |

### 製作步驟

#### Step 1：水平感測電路
1. 將兩顆光敏電阻相距 5cm 固定在水平遮光隔板兩側
2. 每顆光敏電阻串聯一顆 10KΩ 電阻，形成分壓電路
3. 兩組分壓電壓分別輸入 LM393 的 V+ 與 V- 腳
4. 中間的遮光隔板確保只有當太陽在正前方時，兩側光強才相等

#### Step 2：垂直感測電路
製作方式同上，安裝於垂直方向（南北向）

#### Step 3：馬達驅動電路
1. LM393 輸出接到 L293D 的輸入腳位
2. L293D 輸出接到減速馬達
3. LED 接在輸出端作為轉向指示（紅燈→正轉，綠燈→反轉）

#### Step 4：供電系統
1. 使用 6V 電池盒供電（4 顆 AA 電池）
2. 加裝電源開關
3. 可串接二極體防止反接

## 五、追日運作邏輯

```
          太陽在東側                太陽在正上方                太陽在西側
       ┌──────────────┐        ┌──────────────┐        ┌──────────────┐
       │  ☀️           │        │      ☀️       │        │           ☀️  │
       │  │            │        │      │       │        │            │  │
       │ LDR1 LDR2    │        │  LDR1 LDR2   │        │  LDR1 LDR2  │
       │ V+ > V-      │        │ V+ = V-      │        │ V+ < V-     │
       │ 馬達→向東轉  │        │ 馬達→停止   │        │ 馬達→向西轉  │
       └──────────────┘        └──────────────┘        └──────────────┘
```

**判斷條件：**
- V+ > V- → 馬達正轉（向東 / 向上）
- V+ < V- → 馬達反轉（向西 / 向下）
- V+ = V- → 停止（已正對太陽）

## 六、整體組裝流程

```
開始
 │
 ├── ① 完成拋物面太陽爐本體（18片組合）
 │
 ├── ② 完成轉動載臺（底座 + 轉盤 + 齒輪 + 馬達）
 │
 ├── ③ 完成追日電路（麵包板測試通過）
 │
 ├── ④ 將太陽爐固定在載臺的垂直轉動軸上
 │
 ├── ⑤ 安裝水平光敏電流感測器（東西向）於載臺
 │
 ├── ⑥ 安裝垂直光敏電流感測器（南北向）於爐體側邊
 │
 ├── ⑦ 連接電路與馬達，確認轉動方向正確
 │
 ├── ⑧ 裝上黑色鍋具於焦點處（距爐底 10cm）
 │
 ├── ⑨ 戶外測試，調整靈敏度
 │
 └── ⑩ 開始正式實驗記錄
```

## 七、安全注意事項

1. **高溫危險**：焦點處溫度可達 100℃ 以上，實驗時戴隔熱手套，勿將手或臉靠近焦點
2. **用電安全**：所有電路使用低電壓（6V），但避免短路
3. **切割安全**：裁切壓克力與木板時須有成人陪同，使用護目鏡
4. **烤箱安全**：加熱壓克力時須在通風處，避免吸入氣味
5. **太陽安全**：勿直視反射的太陽光，以免傷害眼睛，建議戴太陽眼鏡
6. **雨天停用**：電路與木材怕水，下雨天請勿使用
